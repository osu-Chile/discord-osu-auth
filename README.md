# Discord osu! Auth
Discord bot para verificar la identidad usando su cuenta de osu!

## Requisitos

* osu! OAuth Application (https://osu.ppy.sh/docs/index.html#registering-an-oauth-application)
* MySQL Database
* Apache2
* Python 3.7
